# CRLF
A text stegosystem based on changing the order of the end-of-line markers CR/LF.

# Text Steganography

The method is based on changing the end-of-line markers CR/LF, due to which the message is hidden.

# Running
```
go build -o .
```
```
CRLF.exe
```


# Running debug mode
```
go build -o .
```
```
CRLF.exe --dbg
```
